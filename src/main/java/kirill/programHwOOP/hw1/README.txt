Класс Main - точка входа, имеет внутренние методы покупки товара и поиск по имени товара в категориях.

Класс Goods - класс, описывающий товары, метод setPrice() для наглядности примера, выставления ограничений,
 подобные условия можно описать для кол-ва товара, чтобы оно не было отрицательным,
 и, например, для rate аналогичным образом.

Класс User - представляет собой класс по описанию пользователя

Класс Category - описывает существующие категории товаров, а также имеет статическое поле с названиями всех категорий,
 чтобы предотвратить повторение одинаковых.

Класс Basket - представляет собой словарь пар ключ-значение и используется как корзина товаров, выбранных пользователем.

** 2 ключевых метода в Main: buy() и findGoalGoods();
 первый с помощью второго находит нужный товар по его названию,
 далее проверяется, что содержит в себе результат findGoalGoods(),
 если !null - то происходит проверка соответствия по кол-ву товара в наличии и спросу покупателя,
 если null - получаем сообщение о том, что такого товара не существует.
