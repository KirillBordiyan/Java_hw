Класс Members - родительский класс для Human, Cat, Robot
 имеет в конструкторе имя, длину дистанции бега и высоту прыжка, также поле canParticipate определяет,
 может ли "экземпляр" принимать участие в дальнейшем.
 2 приватных метода для выполнения прыжка или преодоления дистанции.
 Они объеденины в один метод passAnObstacle(), который определяет к какому типу относится препятствие
 и вызывает нужный метод.

Для читаемости, классы Wall и Tradmill - наследники общего класса Obstacle, необходимо это для возможности иметь массив
 с разными типами препятствий

В точке входа описаны 2 массива: участников и препятствий, если участник не проходит препятствие - он выбывает,
 об этом есть вывод информации в консоль
