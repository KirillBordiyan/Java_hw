Точка входа ../app/ProgramHw3.java.

В папке controller находятся контроллер и компонент для записи(или создания + записи) в файл
     внутри также располагается папка exc, которая содержит собственные типы исключений:
        -для фио,
        -для номера телефона,
        -для даты рождения,
        -проверки гендера.

Папка data используется как БД для создаваемых программой файлов (существующие файлы - пример использования программы).

Принцип работы программы:
    -после запуска пользователь должен ввести в консоль информацию, которую у него запрашивают в соответствующем формате;
    -далее введенная строка преобразуется (удаляются случайные лишние пробелы);
    -проверка на соответствие по кол-ву параметров (если меньше, выбрасывается ошибка об этом);
    -последовательно проверяется каждый из требуемых параметров, в случае неверных данных/формата данных, пользователь
     будет уведомлен, что именно не так;
    -далее вся информация передается в компонент для записи, за основу для имени нового (если требуется) файла
     используется "фамилия" из введенных данных;
    -при наличии ошибки во время записи, в коде это учтено, появится сообщение об этом + стактрейс.
